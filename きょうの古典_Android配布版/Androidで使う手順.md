# きょうの古典を Android で使う

この配布版は、Android のホーム画面へ追加して、アプリのように開ける PWA 版です。APK のインストールは不要です。

## おすすめ：ホーム画面アプリとして使う

1. ZIP ファイルを Windows または Mac で展開します。
2. 展開したフォルダを、静的サイト公開サービスへアップロードします。もっとも手軽なのは Netlify Drop です。
   - PC のブラウザで https://app.netlify.com/drop を開く
   - 展開したフォルダの中身（index.html、sw.js、manifest.webmanifest、icon.svg）をアップロードする
   - 表示された公開 URL を控える
3. Android 端末の Chrome で、その公開 URL を開きます。
4. Chrome 右上のメニューから「アプリをインストール」または「ホーム画面に追加」を選びます。
5. ホーム画面に追加された「きょうの古典」を開いてください。

最初に一度オンラインで開けば、アプリ本体は端末に保存されます。履歴・本棚・お気に入りは、その Android 端末と公開 URL ごとに保存されます。

## 友人へ渡す場合

ZIP を渡すだけではホーム画面アプリにはなりません。公開した URL を共有してください。同じ URL を開いた各人の履歴・ストック・お気に入りは、それぞれの端末内で独立して保存されます。

## 更新する場合

新しい配布版を同じ公開先へアップロードしてください。Android 側では、アプリを一度閉じてから再度開き、画面を更新すると新しい内容が反映されます。

## 補足

Android のファイルアプリから index.html を直接開くこともできますが、ブラウザの種類によっては保存機能やホーム画面追加が安定しません。日常的に使う場合は、上記の公開 URL を使う方法をおすすめします。
