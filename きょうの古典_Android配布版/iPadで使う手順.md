# きょうの古典を iPad で使う

Android版と同じ公開 URL を、iPad の Safari で開けば利用できます。追加のアプリや APK は必要ありません。

## ホーム画面へ追加する

1. iPad の Safari で、公開済みの「きょうの古典」の URL を開きます。
2. 画面上部または下部の共有ボタン（四角から上矢印のアイコン）をタップします。
3. 「ホーム画面に追加」を選びます。
4. 名前が「きょうの古典」になっていることを確認し、「追加」をタップします。
5. ホーム画面のアイコンから開いてください。

ホーム画面から起動すると、Safariのタブ表示から離れ、アプリに近い見た目で開きます。最初にオンラインで開いた後は、アプリ本体が端末に保存されます。

## 大切な点

- 履歴・本棚・お気に入りは、iPad ごとに保存されます。
- 同じ公開 URL を使えば、Android と iPad のどちらでも利用できます。ただし履歴などは端末間で同期されません。
- 更新版を公開した後は、アプリを一度閉じてから再度開くと反映されます。

## 公開URLがまだない場合

Android向けの手順と同じく、Netlify Drop などで配布フォルダを公開してください。公開 URL を iPad の Safari で開けば、そのまま上の手順へ進めます。
